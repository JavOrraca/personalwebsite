Network Visualizations via R + iGraph

Network visualizations can be useful for understanding entity and large data interdependencies. I’ve been using R and the iGraph package to explore network analysis and found this tutorial to be EXCELLENT by Katherine Ognyanova. It takes a lot of time to build these network graphs and more time to interpret, but definitely rewarding when it provides insights you may have otherwise overlooked.
